# Оракул трёх слов

Небольшое приложение, которое просит 3 слова и выдаёт образное «мистическое» предсказание без прямого повторения введённых слов.

## Варианты запуска

### 1) Локально через PHP

Требуется установленный PHP 8+.

```bash
php -S 127.0.0.1:8000 index.php
```

Откройте в браузере: <http://127.0.0.1:8000>

### 2) Через GitHub Pages (без PHP)

GitHub Pages не исполняет PHP, поэтому для Pages используется статическая версия `index.html` с той же логикой на JavaScript.

## Публикация на GitHub Pages

1. Запушьте изменения в репозиторий на GitHub.
2. Откройте **Settings → Pages**.
3. В блоке **Build and deployment** выберите:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` (или ваша основная ветка), папка `/ (root)`
4. Нажмите **Save**.
5. Подождите 1-2 минуты, после чего откроется URL вида:
   - `https://<username>.github.io/<repo>/`

## Файлы

- `index.php` — версия для локального запуска через PHP.
- `index.html` — версия для GitHub Pages.
